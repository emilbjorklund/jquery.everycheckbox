Demo of Everycheckbox jQuery plugin

Click the master toggle to toggle all/none of the checkboxes.

Unchecking the a checked box also unchecks the master.

Add data-master-toggle="true" to the element that is the master toggle, or pass in a selector for the element that you want to make the master toggle as the value for the masterSelector option.

Call the plugin on the element enclosing the checkboxes, like so:

$('.check').everycheckbox();
Please select any that apply

This fieldset doesn't have a data-master-toggle marker, but is activated by passing in the ID of the field as a jQuery selector to act as master toggle instead, using this code:

$('.check-custom').everycheckbox({
			masterSelector: '#master-toggle-custom'
		});
